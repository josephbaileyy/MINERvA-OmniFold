# Visual prototypes

Open `index.html` in a browser. It is a self-contained gallery: no server,
network connection, or external JavaScript is required.

These are motion sketches for choosing a visual language, not final physics
figures. All plotted points are deliberately generic toy data with unlabeled
axes. The real result plots remain in `../claude-design-package/figures/`.

## Included concepts

1. The dimensionality wall
2. One OmniFold iteration
3. Bin at the end
4. Marginalization as validation
5. Localizing an excess

Use the arrow keys to step, the spacebar to play/pause, and the number keys
1–5 to switch concepts. `static-overview.svg` shows the five final frames on
one page.

## Palette

- Data: near-black
- Unweighted simulation: blue
- Reweighted simulation: orange
- Validation/reference: teal
- Truth-space accents: violet

The palette is color-vision-friendly at presentation scale and stays fixed
across the prototypes.
