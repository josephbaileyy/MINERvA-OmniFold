The problem

The problem

Twenty-nine of thirty measurements at one or two observables

The 30 published measurements surveyed here use binned D'Agostini iterative Bayesian unfolding (IBU); 29 report one or two simultaneous observables, one is triple-differential (QE-like pT, p∥, ΣTp, 2022). This work unfolds five variables simultaneously, unbinned; point-cloud inputs are an outlook study.

The method

Binned OmniFold gives the D'Agostini update

The method

The only ML ingredient: a classifier as a ratio estimator

p(data | x) — the output of a classifier trained to separate data from simulation (class-normalized)
w(x) = p(data | x)1 − p(data | x)
  • Indistinguishable here → w ≈ 1
  • Data-rich region → w > 1
  • Simulation-heavy region → w < 1
The method

One iteration: reweight, then pull back

The method

What is controlled in the implementation

  • With binned inputs, the update is exactly D'Agostini IBU. The unbinned version retains the simulated truth–reco pairing event by event.
  • The nominal weights come from gradient-boosted decision trees. An independently trained neural network gives a 1.008 total-cross-section ratio (backup B3).
  • Backgrounds, fakes, misses and efficiency are handled with the standard bookkeeping. Details and validation figures in backup.
The method

An added observable becomes an input feature

Binned: bins multiply

Each new dimension multiplies the response matrix: 2 → 4 → 8 → … cells, each needing its own migration statistics. The binning is frozen before the unfold and can never be revisited.

Unbinned: columns add

Each trained observable is one more input column on the same events. The output is a weighted event sample: choose reporting bins and project recorded features after unfolding.

Summary

Summary: weights on events, not bins

  • With binned inputs, OmniFold gives the D'Agostini IBU update; unbinned inputs retain per-event weights.
  • It reproduces the published 2D result, with an independently rebuilt uncertainty budget.
  • The same events then open Eavail, q3 and W — recovering the known low-recoil physics and localizing a residual excess in the DIS corner.

Backup 1 · Algorithm

The OmniFold iteration, precisely

  • Step 1 — at reco level, train a classifier to separate data from (current-weight) simulation. Its output estimates the likelihood ratio data/simulation — the per-event reco weight ω.
  • Step 2 — at truth level, train a second classifier to turn the pulled-back weights into a function of truth variables only. This gives the truth-level weight ν — a proper reweighting of the generator prior.
  • Iterate: ν seeds the next pass. With binned inputs this is exactly D'Agostini IBU. Full derivation and notation in the OmniFold primer.
Backup 1b · From the open data product to a cross section

Four event trees preserve the analysis logic

  • data (4.1M selected events) · mc_background · mc_signal_reco (paired reco+truth) · mc_truth_denom (32.8M, authoritative). Truth-only events enter as native misses; 12 ME playlists run separately, then merge.
  • Step-1 target: phase-space data with a non-negative per-bin purity weight (data − background)/data. An explicitly unbinned negative-weight subtraction reproduces it (backup B2).
  • d²σ = Σ wpush·wtruth / (Φ · NPOT · Nnucleons · ΔpT · Δp∥) per bin. Misses make the truth ensemble acceptance-corrected: completeness = 1, no second efficiency division.

Backup 4 · Regularization

Iteration count and classifier capacity

  • The iteration count plays the same regularizing role as in D'Agostini IBU. Headline results use five iterations — the "5iter" in every figure name.
  • Classifier capacity (tree depth, count, learning rate) bounds how sharp a reweight can be learned. The chosen GBDT capacity is tested against an independently trained neural network (B3).
  • Stability is verified directly: seed and train-split scans, next slide.
Backup 5 · Seed & split variability (2D — cleared)

Backup 6b · Anatomy of the χ² = 3.66

Per-bin differences and correlated shape are different questions

  • Published-σ standardized differences: mean 0.089, rms 0.60 — every bin lies within one published σ.
  • Indicative distance using the paper's full covariance: 3.66 per bin. Driven by correlated shape modes along the low-pT peak ridge — not normalization (total σ ratio 1.011).
  • Using the sum of the paper and OmniFold covariances: 1.48 per bin (diagnostic only). The results share data and their cross-covariance is unavailable; a formal difference test would require it.
