### 2. Sizes, Footing, and Overlap
*   **Sizes**: The 10 standard files are exactly ~53.8 GB each. The 10 FPS files are exactly ~74.8 GB each. Total I/O required is ~1.28 TB.
*   **Immutability**: Both directories are fully populated, and all event loop / merge writers have terminated. The files are immutable. No `squeue` jobs are running.

*   **Writers**: No current process is writing to these files. They are completely safe to read.

### 3. Batch Layout & Atomic Publication
*   **Hardware/QOS**: 1 node, `shared` QOS, 4 CPUs, 8 GB memory. We bound parallelism to `xargs -P 4` to saturate the node-local I/O bandwidth from Lustre (typically peaking around 1.5–2 GB/s per node) without overwhelming metadata servers.
*   **Wall Time**: At a conservative 400 MB/s, reading 1.28 TB takes ~1 hour. A `03:00:00` wall time safely pads this.
*   **Collision-Free Output**: Outputs go to the orchestrator-owned `docs/orchestration/state/` directory to strictly avoid writing inside A or C’s `active_universe_5d` namespaces.
*   **Atomicity**: Hashes are written to `.tmp` files and `mv`'d to their final `.txt` paths only on success.

### 4. Owner-Neutral Launcher Contract
No existing committed launcher does this task. The orchestrator must add the smallest possible contract script.

Create `docs/orchestration/sbatch_hash_merged_inputs.sh`:
```bash
#!/bin/bash
#SBATCH -A m3246
#SBATCH -C cpu
#SBATCH -q shared
#SBATCH -n 1
#SBATCH -c 4
#SBATCH --mem=8G
#SBATCH -t 03:00:00
#SBATCH -J hash_merged
#SBATCH -o docs/orchestration/state/hash_merged-%j.out

set -eo pipefail
cd /pscratch/sd/j/josephrb/MINERvA-OmniFold

mkdir -p docs/orchestration/state
std_out="docs/orchestration/state/standard_merged_sha256.txt"
fps_out="docs/orchestration/state/fps_merged_sha256.txt"

date -u "+[%Y-%m-%dT%H:%M:%SZ] Hashing standard merged ROOTs..."
find nd-unfolding/active_universe_5d/standard/merged -name "runEventLoopOmniFold_5D_MEFHC_active_*.root" -print0 | \
  xargs -0 -n1 -P4 sha256sum > "${std_out}.tmp"
mv "${std_out}.tmp" "${std_out}"

date -u "+[%Y-%m-%dT%H:%M:%SZ] Hashing FPS merged ROOTs..."
find nd-unfolding/active_universe_5d/fps/merged -name "runEventLoopOmniFold_5D_FPS_active_*_universes_full.root" -print0 | \
  xargs -0 -n1 -P4 sha256sum > "${fps_out}.tmp"
mv "${fps_out}.tmp" "${fps_out}"

date -u "+[%Y-%m-%dT%H:%M:%SZ] Done."
```
